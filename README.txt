Application: My application is a Math game in which the player must answer randomly generated mathematical equations.

Instructions on how to run application:
The GameClient class contains an IP address of "localhost" and a port number of 3500, while the GameServer is contains a port number of 3500.
To run the game, 

1. compile the .java files using "javac <filename>.java", 

2. on one terminal, run "java GameServer" and

3. on a seperate terminal, run "java GameClient".

Follow the game prompts and enjoy...

NOTE: Do not include symbols in your answers except "." for decimals!!!
